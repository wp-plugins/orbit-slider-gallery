== Description ==
Image slider based on orbit js

== Installation ==

Upload wp_orbit_gallery directory to the /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress
Create new orbit gallery through the 'Orbit Sliders' menu in WordPress 3.Customize setting through the 'Orbit Sliders Settings' menu in WordPress
Place <?php [orbit_slider id='slider_id'] ?> in your post or page
