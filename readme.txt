== Description ==
Orbit_Gallery is based on jQuery Orbit Slider from the team Zurb. 
It uses a custom post type for each slide. 
Slider settings are customizable through plugin settings page. 
== Installation ==

1. Upload wp_orbit_gallery directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload media through the 'Media' menu in WordPress
4. Create new orbit gallery through the 'Orbit Sliders' menu in WordPress	 
5. Customize setting through the 'Orbit Sliders Settings' menu in WordPress
6. Place <?php [orbit_slider id='slider_id'] ?> in your post or page
